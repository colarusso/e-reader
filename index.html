This is a very simple eReader for text files created by Colarusso. Just specify the file you want to view by appending a `title` parameter to the URL with the path and name of a .txt file (e.g., ?title=texts/story_name). Keep in mind, however, that it can only read files on the same domain as this page. If you want to use adapt or use this reader, you can find the code on GitHub.
